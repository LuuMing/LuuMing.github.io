# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Is

Static GitHub Pages site for **LuuMing.github.io**. No build system, no bundler, no package manager — all files are served as-is.

## Architecture

- **`index.html`** — Main landing page: developer portfolio / app showcase (hand-written, self-contained HTML+CSS). This is the primary file you'll be editing.
- **`blog.html`** — Legacy Hexo-generated blog index (NexT theme, Hexo 3.9.0). Treat as static output — do not hand-edit.
- **`YYYY/MM/DD/*/index.html`** — Hexo-generated blog post pages (2017–2020). Also static output.
- **`archives/`, `categories/`, `tags/`, `page/`** — Hexo-generated listing pages.
- **`css/`, `js/`, `lib/`** — Static assets for the Hexo blog theme.
- **`app-ads.txt`** — Google AdMob app-ads.txt for ad verification.

## Key Points

- There is no build step. To preview, open HTML files directly in a browser or use any local HTTP server (`python -m http.server`, etc.).
- The blog portion (`blog.html` and dated post directories) was generated by Hexo and is no longer actively maintained. New content goes into `index.html`.
- `index.html` is a single-file app with inline CSS — styles and markup live together. No external stylesheet.
- The site language is Chinese (zh-CN).
